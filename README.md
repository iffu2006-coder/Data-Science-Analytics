## Completed Task
### Task 1: Exploring and Visualizing Iris Dataset
## Objective
The objective of this task is to understand how to read, summarize, and visualize a simple dataset.
## Dataset Used
Iris Dataset
## Tools and Libraries Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn
## Work Done
In this task, I loaded the Iris dataset using pandas. I checked the structure of the dataset using shape, columns, and head functions. I also explored the dataset using info and describe functions.
After that, I created basic visualizations including:
- Scatter plot
- Histogram
- Box plot
## Conclusion
The Iris dataset was successfully explored and visualized. The visualizations helped in understanding the relationship between different flower measurements. Petal length and petal width were useful features for separating different iris species.
